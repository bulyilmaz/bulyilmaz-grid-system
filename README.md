# Grid System with SASS

## Installation


```bash
npm install --save bulyilmaz-grid-system
```

## Breakpoints

```bash
sm: 576px

md: 768px

lg: 992px

xl: 1200px
```

## Usage

```html
<head>
    <link rel="stylesheet" href="style.css">
</head>

<body>
    <div class="row">
        <div class="col-lg-6 col-md-12 col-sm-12"></div>
        <div class="col-lg-3 col-md-6 col-sm-12"></div>
        <div class="col-lg-3 col-md-6 col-sm-12"></div>
    </div>
</body>
```
